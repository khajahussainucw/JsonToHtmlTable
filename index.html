JSON in, HTML <table> out.

A small .NET library that converts any JSON — objects, arrays, deeply nested structures — into a clean, XSS-safe HTML table. Built on System.Text.Json, targets netstandard2.0, no Newtonsoft.Json dragged into your project.

dotnet add package JsonToHtmlTable

An object becomes a two-column table

Pass a JSON object in, get a Key / Value table back. Nested objects render as inner tables, arrays render as inner tables — recursively, all the way down.

C#
using JsonToHtmlTable;

string json = """
{
    "subscriberId": "SUB-100245",
    "msisdn": "+91-9876500000",
    "planName": "Postpaid Pro 599",
    "isActive": true,
    "balance": 412.75,
    "tags": ["VoLTE", "5G", "Roaming"],
    "address": {
        "circle": "APAC-IN",
        "billingZone": "ZN-7"
    }
}
""";

string html = JsonToHtmlConverter.Convert(json);

An array of records becomes a grid

One column per field, one row per record. If records have different keys, the columns are the union — missing cells fall back to your NullText.

C#
string cdrs = """
[
    { "callId": "C-001", "originator": "9876500000", "destination": "9123400000", "durationSec":  73, "billedAmount": 1.21 },
    { "callId": "C-002", "originator": "9876500001", "destination": "9123400022", "durationSec":  14, "billedAmount": 0.30 },
    { "callId": "C-003", "originator": "9876500099", "destination": "9123400088", "durationSec": 412, "billedAmount": 6.85 }
]
""";

string html = JsonToHtmlConverter.Convert(cdrs);

Why pick this one

No Newtonsoft baggage

Built on System.Text.Json. Installing it doesn't drag a JSON.NET version into your project graph.

Runs basically everywhere

Targets netstandard2.0 — old .NET Framework apps, modern .NET 10, Xamarin, Unity, the works.

XSS-safe by default

Every value and key passes through HtmlEncoder. International characters preserved; dangerous ones escaped.

Handles weird JSON

Top-level primitives, mixed-type arrays, arrays of arrays, deep nesting, big numbers, Unicode — covered.

Streams big payloads

TextWriter overloads write directly to a file or HTTP response — no intermediate string allocation.

Doesn't throw on bad input

TryConvert returns false for invalid or empty JSON, so user-supplied input never crashes you.

Customize what's emitted

Default output is a bare <table class="json-to-html-table">…</table> with no inline styling — meant for pages that already have a stylesheet. Flip InlineStyles to get borders and padding baked in for places CSS won't reach (HTML email, Notion, quick test pages). 

var html = JsonToHtmlConverter.Convert(json, new HtmlTableOptions
{
    TableCssClass        = "table table-striped",
    KeyHeader            = "Field",
    ValueHeader          = "Data",
    PrimitiveArrayHeader = "Tag",
    NullText             = "—",
    ShowRowNumbers       = true,
    KeyTransform         = key => Regex.Replace(key, "([a-z])([A-Z])", "$1 $2"),
    InlineStyles         = true,
    MaxDepth             = 32,
});

The whole API surface

One static class, a handful of overloads, one options bag. Resist the urge to look for more. 

public static class JsonToHtmlConverter
{
    public static string Convert(string json, HtmlTableOptions? options = null);
    public static void   Convert(string json, TextWriter writer, HtmlTableOptions? options = null);

    public static string Convert(JsonDocument document, HtmlTableOptions? options = null);
    public static string Convert(JsonElement element,  HtmlTableOptions? options = null);
    public static void   Convert(JsonElement element,  TextWriter writer, HtmlTableOptions? options = null);

    public static string ConvertObject(
        object? value,
        HtmlTableOptions? options = null,
        JsonSerializerOptions? serializerOptions = null);

    public static bool TryConvert(string? json, out string html, HtmlTableOptions? options = null);
}
